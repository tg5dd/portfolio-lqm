# 梁秦铭 — 个人主页

赛博未来感风格的个人主页,展示网络工程 / 软件测试相关的经历与项目。

## 技术栈

- React 18 + TypeScript
- Vite 5
- React Router (HashRouter)
- 原生 CSS + CSS 变量

## 本地开发

```bash
pnpm install
pnpm dev          # http://localhost:5173/
pnpm build        # 生成 dist/
pnpm preview      # 预览生产构建
```

## 部署

通过 GitHub Actions 自动部署到 GitHub Pages。推送到 `main` 分支即触发。
